Subschema Demo
===
The Demo app for Subschema

##Demo
See it in action [here](http://subschema.github.io/subschema)

Or run it 

```sh
  git clone https://github.com/subschema/subschema-demo.git
  cd subschema-demo
  npm install
  npm run hot &
  open http://localhost:8082
```

##Installation
```sh
 $ npm install subschema-image
``
